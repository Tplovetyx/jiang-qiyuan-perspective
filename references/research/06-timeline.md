# 06-timeline.md — 生平时间线与时代背景

## 个人时间线

| 年份 | 事件 |
|------|------|
| 1956 | 考入清华大学电机系电子计算机专业 |
| 1959 | 大三时被抽调进入数学师资培训班，从电机系转向数学 |
| 1962 | 从基础课数学培训班毕业，留校在数学教研组任教 |
| 1982 | 首次接触"数学建模"思想 |
| 1983 | 接替萧树铁，在清华首次开设数学建模课程 |
| 1984 | 正式主讲数学建模课程，持续三十余年 |
| 1987 | 主编《数学模型》第1版出版（国内首部） |
| 1992 | 参与发起全国大学生数学建模竞赛 |
| 1994 | 担任首届全国组委会秘书长（至2008年） |
| 1997 | 获国家级教学成果二等奖 |
| 1998 | 在清华首次开设数学实验课程 |
| 1999 | 《数学实验》出版（国内最早之一） |
| 2001 | 再次获国家级教学成果二等奖 |
| 2005 | 退休；数学实验课程获评国家精品课程 |
| 2008 | 卸任竞赛组委会秘书长，改任专家组副组长 |
| 2013 | 完全退出竞赛组委会工作 |
| 2024 | 《数学模型》第六版出版，累计印数超140万册 |

## 时代背景交汇

- **1950年代**：新中国优先发展重工业和国防科技，电计算机专业是前沿方向之一
- **1978年后**：改革开放，经济建设需求推动应用数学发展
- **1980年代**：个人计算机普及的关键时期，数值计算成为可能
- **1990年代**：高校教育改革浪潮，素质教育理念推动数学建模课程化
- **竞赛发展**：从1992年300多队，发展到2013年23000+队
